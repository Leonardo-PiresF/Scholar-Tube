# ScholarTube

## Descrição do Projeto
O **ScholarTube** é uma plataforma de streaming de vídeos voltada para estudantes e criadores de conteúdo educacional. O projeto busca oferecer uma experiência de usuário intuitiva, permitindo explorar, assistir e organizar vídeos de forma prática, similar a plataformas como YouTube, mas focada em conteúdo acadêmico e de aprendizagem.

O site conta com funcionalidades como:
- Tela principal com feed de vídeos.
- Player de vídeo integrado.
- Sistema de organização e avaliação de conteúdo.
- Design responsivo e moderno, pensado para aprendizado e produtividade.

## Tecnologias Usadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:
- **HTML5** e **CSS3** para estrutura e estilo.
- **JavaScript** para interatividade.
- **Figma** para design e prototipagem.
-**git/github** para versionamento do projeto.
